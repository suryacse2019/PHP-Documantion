Important files:
1. .ini :initialisation files
		it will contain all the important services , assets, resource,permission , utilities required by any program during startup.
		Example:
			php.ini
			laragon.ini
			mysql.ini
			etc etc..
			
2. .log : log files : it will contain all the run time error and exception occured during execution of the Program
	it restore the information with alerts status 
		1.success
		2.info
		3.warning
		4.danger(critical)
